# FRR iBGP EVPN

За основу взят конфиг из [репозитория](https://github.com/fedepaol/evpnlab)

## Схема сети

```raw




                            ┌─────────┐
                            │         │
                            │ spine   │
                            │         │
                            └────┬────┘
                                 │
                                 │
                         ┌───────┴────────┐
                         │                │
                    ┌────┴────┐      ┌────┴────┐
                    │         │      │         │
                    |  leaf1  │      │  leaf2  │
                    │         │      │         │
                    └──┬──┬───┘      └────┬─┬──┘
                       │  │               │ │
              ┌────────┘  │               │ │
              │   L2      │ L2         L2 │ └──────────┐ L2
              │           │               │            │
        ┌─────┴────┐  ┌───┴──────┐  ┌─────┴────┐  ┌────┴─────┐
        │          │  │          │  │          │  │          │
        │ HostL2_1 │  │ HostL2_2 │  │ HostL2_2 │  │ HostL2_1 │
        │          │  │          │  │          │  │          │
        └──────────┘  └──────────┘  └──────────┘  └──────────┘
```

## Запуск

Для запуска необходимо выпонить
```
bash setup.sh
```

Если в вас установлен Docker, то необходимо убрать ключ --runtime и его аргумент в скрипте, и заменить podman на docker

Лаба запускается быстро, но сходится долго.
